# RAFT

A Raft implementation in Go.

## Getting Started

Clone this project into your `$GOROOT`

```
cd $GOROOT
git clone https://github.com/carlos-lopez-garces/raft.git
```

### Prerequisites

This project needs at least Go 1.9
